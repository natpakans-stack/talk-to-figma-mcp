มันคืออะไร?

Talk To Figma MCP Plugin คือระบบที่ทำให้ Claude AI สามารถ:

  • สร้าง Frame, Rectangle, Text, Shape ใน Figma ได้
  • แก้ไขสี, ขนาด, ตำแหน่ง, Auto Layout ของ element ที่มีอยู่แล้วได้
  • อ่านข้อมูล document, selection, styles, components ได้
  • Clone, Delete, Export element ได้
  • ใช้ Components จาก local library ได้

ทำไมถึงน่าใช้?

สิ่งที่ทำได้ตัวอย่าง
สร้าง UI จากคำอธิบาย"สร้างหน้า Login แบบ iOS ให้หน่อย"
แก้ไข design ที่มี"เปลี่ยนปุ่มเป็นสีน้ำเงิน, เพิ่ม corner radius 12"
สร้าง Layout ซ้ำๆ เร็ว"สร้าง card 10 ใบ เรียงกัน"
ออกแบบตาม Design System"สร้างหน้าจอตาม Material Design 3"

ข้อจำกัด

  • ไม่สามารถสร้าง Vector/Pen tool ซับซ้อนได้
  • ไม่สามารถ import รูปภาพจาก URL ได้โดยตรง
  • ไม่สามารถใช้ Prototype/Interaction ได้
  • ต้องเปิด Figma Desktop (ใช้ผ่าน browser ไม่ได้)
