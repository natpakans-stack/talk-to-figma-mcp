มันคืออะไร?

HTML-to-Figma Workflow คือระบบที่ทำให้ Claude AI สามารถ:

  • สร้าง HTML/CSS preview แล้ว import เข้า Figma เป็น layers อัตโนมัติ
  • อ่าน design ที่มีอยู่แล้ว — ถ่ายภาพหน้าจอ, ดู design context, อ่าน metadata
  • Iterate เร็ว — แก้ HTML ใน Claude → refresh browser → เห็นผลทันที
  • สร้างทั้งหน้าจอพร้อมกัน — ไม่ต้องสร้างทีละ element
  • ได้ layout ที่แม่นยำ — CSS → Figma Auto Layout mapping

ใช้ MCP 2 ตัว

MCP Serverหน้าที่ตัวอย่าง
Figma MCP (อ่าน)อ่าน design จาก Figmaถ่ายภาพหน้าจอ, ดู specs, อ่าน metadata
html-to-design MCP (เขียน)ส่ง HTML ไปสร้างใน Figmaimport_html, import_url

ข้อจำกัด

  • ต้องเปิด Figma Desktop + html.to.design plugin ค้างไว้ตอน import
  • Font อาจ map ไม่ตรง 100% — ต้อง fine-tune ใน Figma หลัง import
  • ไม่สามารถสร้าง Prototype/Interaction ได้
  • HTML ที่ใช้ absolute positioning อาจแปลงไม่ตรง — ใช้ flexbox จะดีกว่า
