5 ขั้นตอนใช้งานรายวัน

ทำทุกครั้งที่เปิดเครื่องใหม่ หรือทุกครั้งที่ต้องการใช้งาน

เปิด html.to.design Plugin ใน Figma

  1. เปิด Figma Desktop
  2. เปิดไฟล์ที่จะทำงาน
  3. คลิกขวาที่ canvas → Plugins → html.to.design
  4. ไปที่ tab MCP
  5. ตรวจว่า toggle "Enable MCP endpoint" เปิดอยู่
  6. ตรวจว่า STATUS: connected (จุดเขียว)
  ┌──────────────────────────────┐
  │  html.to.design              │
  │                              │
  │  Tab: MCP                    │
  │  MCP endpoint: ON            │
  │  STATUS: connected ✓         │
  └──────────────────────────────┘
⚠️
สำคัญ: Plugin ต้องเปิดค้างไว้ตลอดการใช้งาน — ถ้าปิด plugin จะ import HTML ไม่ได้

เปิด Claude แล้วเริ่มใช้งาน

  1. เปิด VS Code
  2. เปิด Claude Code panel (icon Claude ที่แถบซ้าย)
  3. ทดสอบว่าเชื่อมต่อได้ โดยพิมพ์:
สร้างปุ่มสีน้ำเงิน แล้ว import เข้า Figma

4. ถ้าเชื่อมต่อสำเร็จ Claude จะ:

  1. สร้าง HTML/CSS สำหรับปุ่ม
  2. ใช้ import_html ส่ง HTML ไป Figma
  3. Figma สร้าง frame ที่มี button พร้อม Auto Layout 🎉

Workflow การออกแบบ

ทุกครั้งที่ต้องการสร้าง design ใหม่ ใช้ flow นี้:

  1. บอก Claude ว่าต้องการออกแบบอะไร (เช่น "ออกแบบหน้า Login สำหรับ iOS")
  2. Claude สร้าง HTML preview (.html) → เปิดดูใน browser
  3. Review + Iterate: บอก Claude แก้ไข → Claude แก้ HTML → refresh browser
  4. พอใจแล้ว → บอก Claude: "import เข้า Figma"
  5. Claude ส่ง HTML ผ่าน import_html → html.to.design แปลงเป็น Figma layers
  6. Fine-tune ใน Figma (font, sizing, components)
⚡
Tip: การ iterate ใน HTML เร็วกว่าแก้ใน Figma มาก — แก้ HTML → refresh browser → เห็นผลทันที
