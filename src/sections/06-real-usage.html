6.1 — พื้นฐาน: สั่ง Claude สร้าง Design

เมื่อเชื่อมต่อเสร็จแล้ว สามารถสั่งงาน Claude ด้วยภาษาธรรมชาติ:

พิมพ์Claude จะทำ
"สร้าง frame ขนาด iPhone 15 Pro"สร้าง frame 393×852 px
"สร้างปุ่ม Login สีน้ำเงิน"สร้าง frame + text + สี + radius
"ดูว่าตอนนี้เลือกอะไรอยู่"อ่าน selection แล้วบอก
"เปลี่ยนสีพื้นหลังเป็น #F5F5F5"set fill color ให้ element ที่เลือก
"จัด auto layout แนวตั้ง spacing 16"ตั้ง auto layout ให้ frame ที่เลือก

6.2 — สร้างหน้าจอจาก Spec

สามารถบอก Claude ให้ออกแบบหน้าจอทั้งหน้าได้:

สร้างหน้า Login สำหรับ iOS ใน Figma ให้หน่อย ประกอบด้วย:
- Logo ด้านบน
- Text field สำหรับ Email
- Text field สำหรับ Password
- ปุ่ม Log In สีน้ำเงิน
- ลิงก์ "Forgot Password?"
- ปุ่ม Sign Up ด้านล่าง

Claude จะ:

  1. สร้าง main frame ขนาด iPhone
  2. สร้างแต่ละ component ทีละชิ้น
  3. จัด layout ด้วย auto layout
  4. ตั้งสี, font, spacing ให้เหมาะสม

6.3 — แก้ไข Design ที่มีอยู่

ดู element ที่เลือกอยู่ แล้วเปลี่ยนสีเป็น #007AFF
เพิ่ม corner radius 16 ให้ frame ที่เลือก
เปลี่ยน text ของปุ่มจาก "Submit" เป็น "ส่งข้อมูล"
