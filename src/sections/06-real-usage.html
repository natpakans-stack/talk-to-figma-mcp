6 วิธีใช้งานจริง

สั่ง AI ออกแบบแล้ว import เข้า Figma

6.1 — พื้นฐาน: สั่ง Claude สร้าง Design

เมื่อเชื่อมต่อเสร็จแล้ว สามารถสั่งงาน Claude ด้วยภาษาธรรมชาติ:

พิมพ์Claude จะทำ
"สร้างปุ่ม Login สีน้ำเงิน แล้ว import เข้า Figma"สร้าง HTML ปุ่ม → ส่ง import_html → Figma สร้าง frame
"ออกแบบหน้า Login สำหรับ iOS"สร้าง HTML preview ทั้งหน้า → เปิดดูใน browser
"ดู design จากลิงก์นี้ [URL]"ใช้ get_screenshot ถ่ายภาพ → วิเคราะห์ design
"import เข้า Figma เลย"ส่ง HTML ผ่าน import_html → Figma สร้าง layers

6.2 — สร้างหน้าจอจาก Spec

สามารถบอก Claude ให้ออกแบบหน้าจอทั้งหน้าได้:

ออกแบบหน้า Login สำหรับ iOS ใน Figma ให้หน่อย ประกอบด้วย:
- Logo ด้านบน
- Text field สำหรับ Email
- Text field สำหรับ Password
- ปุ่ม Log In สีน้ำเงิน
- ลิงก์ "Forgot Password?"
- ปุ่ม Sign Up ด้านล่าง

Claude จะ:

  1. สร้าง HTML/CSS preview ทั้งหน้า
  2. ให้เปิดดูใน browser เพื่อ review
  3. ถ้าต้องการแก้ → บอก Claude → แก้ HTML → refresh browser
  4. พอใจแล้ว → บอก Claude "import เข้า Figma"
  5. html.to.design แปลง HTML → Figma layers พร้อม Auto Layout, สี, text ครบ

6.3 — อ่าน Design ที่มีอยู่

ส่ง Figma URL ให้ Claude อ่าน design ได้:

ดู design จากลิงก์นี้แล้วให้ feedback:
https://figma.com/design/xxxxx/MyDesign?node-id=1-2

Claude จะใช้ Figma MCP:

  • get_screenshot — ถ่ายภาพหน้าจอ design
  • get_design_context — อ่านรายละเอียด (layout, colors, fonts)
  • วิเคราะห์แล้วให้ feedback / สร้าง variant ใหม่

6.4 — Iterate แล้ว Import

จุดเด่นของ workflow นี้คือการ iterate ใน HTML ก่อน แล้วค่อย import:

"เปลี่ยนสีปุ่มเป็นสีเขียว แล้วเพิ่ม border radius"
"เพิ่ม dark mode toggle"
"ดีแล้ว import เข้า Figma เลย"
⚡
Tip: Iterate ใน HTML ได้เร็วกว่าแก้ใน Figma — แก้ HTML → refresh browser → เห็นผลทันที
