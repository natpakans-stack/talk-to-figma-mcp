เคล็ดลับสำหรับ Designer

สำหรับ Designer

  1. เลือก element ก่อนสั่ง — ถ้าอยากแก้ไข element ที่มีอยู่ ให้เลือก (click) element นั้นใน Figma ก่อน แล้วค่อยสั่ง Claude
  2. บอก Claude ให้ละเอียด — ยิ่งบอกละเอียด ยิ่งได้ผลลัพธ์ตรงใจ
  3. ใช้ชื่อสีเป็น hex — บอก #007AFF ดีกว่าบอก "สีน้ำเงิน"
  4. บอกขนาดเป็นตัวเลข — บอก "width 393 height 852" ดีกว่า "ขนาด iPhone"
  5. ทำทีละ step — สร้าง frame ก่อน → ใส่ element → จัด layout → ตั้งสี

Workflow แนะนำ

1. ร่าง concept → 2. Claude สร้าง structure → 3. Claude ใส่ content → 4. Review & feedback → 5. Fine-tune ด้วยมือ

สิ่งที่ควรทำเองใน Figma (MCP ทำไม่ได้ / ทำได้ไม่ดี)

  • วาง Image / Illustration
  • สร้าง Complex Vector / Icon (มีเฉพาะ rectangle — ไม่มี ellipse, polygon, star)
  • ตั้ง Prototype & Interaction
  • Fine-tune pixel-perfect alignment
  • สร้าง Complex Gradient / Shadow / Blur (ไม่มี set_effects)
  • สร้าง/ลบ/เปลี่ยนชื่อ Page
  • Group/Ungroup nodes
  • แก้ font family/style หลังสร้าง text (ต้องตั้ง fontSize/fontWeight/fontColor ตอน create_text)
  • ตั้ง line height, letter spacing, text decoration
  • Rename node
